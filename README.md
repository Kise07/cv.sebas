# Portfolio

Personal portfolio website built with Next.js, React, and Tailwind CSS.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Features

- Responsive design
- Projects showcase
- Skills section
- Contact information

## Deployment

Hosted on [Vercel](https://vercel.com) → [cv.yean.me](https://cv.yean.me)
